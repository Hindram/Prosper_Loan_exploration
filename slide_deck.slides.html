Prosper Loan Data Exploration¶

Dataset Overview¶

The data set contains 113,937 records of loans along with 81 variables on each loan. After dropping some irrelevant features of our investigation, we have only 46 variables. The data set includes 31 float, 4 int, and 11 object variables.

Investigation Overview¶

The main features we will investigate in this project are associated directly with answering the following questions:

  • What factors affect users ability to get a loan?
  • What factors impact a high prosper loan risk?
  • What features affects the borrowers APR or interest rate?
  • Are there differences between loans depending on how large the original loan amount was?

These answers associated with many features but mainly 'CreditGrade', 'Term','BorrowerAPR', 'InterestRate', 'ProsperRatingAlpha', 'LoanStatus' and 'LoanOriginalAmount'.
